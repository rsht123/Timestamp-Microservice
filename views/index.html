Timestamp Microservice API

User Stories:

You can pass a string as parameter.
It will check to see if it is a date or a unix time and will accordingly return both the Unix timestamp and a human readable form of that date.

Example Usage:

http://localhost:3000/12 january, 2017
http://localhost:3000/1484179200

Example Output:

{ "unix": 1484179200, "natural": "12 January, 2017" }
